dokku-github-updater

Configuration for GitHub

1. Open the "Add webhooks" page of Your GitHub project.

Firstly, open the hook page of Your GitHub project.
e.g. https://github.com/YOUR_NAME/YOUR_PROJECT_NAME/settings/hooks

Then, click the "Add webhook" button on the top right corner.

2. Fill the form

Then, fill the fields as the follows.

  • Payload URL:

    Hint: You need replace DOKKU_APP_NAME with your published app name on the dokku.
  • Content type:
    application/json
  • Secret
    (Paste the secret string that generated in Initial Setup of dokku-github-updater)
  • Which events would you like to trigger this webhook?
    Just the push event.
  • Active
    True

After that, click the "Add webhook" button on the bottom of the form.

3. Let's trying the hook

You can check the status on the webhook page that configured in the step-2.
